在这个任务中，您需要按照以下步骤将纯文本转换为嵌套密码字典。这种转换与Cipher Dict (decryption)任务中的转换相反。

  1. 将给定的纯文本编码为 utf-8。

  2. 将此值转换为十六进制数，然后将结果转换为十进制数。(该数字的每一位数都将是某个嵌套字典中的密钥）。

  3. 按严格的升序/降序将十进制数分割成多个区块（同一区块内不允许重复相同的数字，否则将启动一个新的区块）。区块顺序应严格交替，第一个区块必须是升序。如果需要保持交替顺序，一个区块可以包含一个数字。

  4. 现在，您需要填充嵌套字典。每个独立的数字块都包含同一个字典的键，值总是空字典。从左到右依次输入数据块，创建嵌套字典，这样每一个下一个数据块的数字都会成为下一个（按创建顺序）空字典的键。请注意区块的顺序和每个区块内数字的顺序。

上面的描述可能会让人感到困惑，让我们看看示例。

输入：纯文本字符串。

输出：嵌套字典形式的密码数据。
