In dieser Mission müssen Sie einen Klartext in ein verschachteltes Chiffrierwörterbuch umwandeln, wobei Sie die folgenden Schritte befolgen müssen. Diese Umwandlung ist das Gegenteil der Umwandlung in der Cipher Dict (decryption) Mission.

  1. Kodieren Sie den angegebenen Klartext in utf-8.

  2. Konvertieren Sie diesen Wert in eine Hexadezimalzahl und wandeln Sie das Ergebnis dann in eine Dezimalzahl um. (Jede Ziffer dieser Zahl ist ein Schlüssel in einer Ebene des verschachtelten Wörterbuchs)

  3. Teilen Sie die Dezimalzahl in Blöcke mit strikt aufsteigender/absteigender Reihenfolge auf (Wiederholungen derselben Ziffer im selben Block sind nicht erlaubt, stattdessen wird ein neuer Block begonnen). Die Reihenfolge der Blöcke sollte streng abwechselnd sein und der erste Block muss aufsteigend sein. Ein Block kann eine Ziffer enthalten, wenn dies notwendig ist, um die abwechselnde Reihenfolge einzuhalten.

  4. Jetzt müssen Sie das verschachtelte Wörterbuch füllen. Jeder einzelne Ziffernblock enthält Schlüssel desselben Wörterbuchs, die Werte sind immer leere Wörterbücher. Indem Sie die Blöcke von links nach rechts nehmen, erstellen Sie ein verschachteltes Wörterbuch in der Weise, dass die Ziffern des jeweils nächsten Blocks zu Schlüsseln im nächsten (in der Reihenfolge der Erstellung) leeren Wörterbuch werden. Achten Sie auf die Reihenfolge der Blöcke und die Reihenfolge der Ziffern innerhalb der einzelnen Blöcke.

Die Beschreibung mag verwirrend sein, also sehen wir uns das Beispiel an.

Eingabe: Ein einfacher Text als Zeichenkette.

Ausgabe: Verschlüsselte Daten in Form eines verschachtelten Wörterbuchs.
